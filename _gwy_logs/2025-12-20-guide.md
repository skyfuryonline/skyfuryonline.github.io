### 欢迎开启 2026 备考之旅！

这是您的备考记录系统 **2.0 版本** 的完整使用说明书。本系统旨在通过**精细化的数据统计**和**可视化的反馈**，辅助您科学备考。

---

#### 1. 如何记录？(核心)

您只需在 `_gwy_logs` 目录下创建一个新的 `.md` 文件（建议用日期命名），并填入以下内容。

**推荐模板 (直接复制)**:

```yaml
---
date: 2026-03-15 20:00:00
title: "今天攻克了图形推理"
subjects:
  - name: 资料分析
    hours: 2
  - name: 判断推理
    hours: 3.5
  - name: 言语理解
    hours: 0
  - name: 数量关系
    hours: 0
  - name: 申论
    hours: 1.5
  - name: 其他
    hours: 0
tags: [刷题, 复盘]
---

### 📝 今日复盘

今天重点复习了图形推理，发现自己对“一笔画”问题还是不够敏感...
```

**字段说明**:
*   **subjects**: 這是核心统计源。请在对应的科目下填写小时数。
*   **tags**: 仅作为标记（如“模考”、“听课”），**不**参与时长统计。

---

#### 2. 智能归类与映射系统

为了方便记录，系统内置了**关键词映射**功能。您在 `name` 字段中可以使用简称，系统会自动归类到标准大类。

| 您填写的 (简称) | 系统归类为 (标准大类) | 饼图颜色 |
| :--- | :--- | :--- |
| **资料**、速算 | **资料分析** | 🔵 蓝色 |
| **图推**、逻辑、类比 | **判断推理** | 🟣 紫色 |
| **言语**、片段阅读 | **言语理解** | 🟢 绿色 |
| **数学**、数量 | **数量关系** | 🟠 橙色 |
| **常识** | **常识判断** | 🟢 青色 |
| **大作文**、公文 | **申论** | 🔴 红色 |
| 面试、模拟 | 面试/模考 | 🟡 黄色/深蓝 |

*建议：尽量使用标准大类名称，或者固定的简称，以保持习惯一致。*

---

#### 3. 可视化图表解读

*   **月历卡片 (中间)**: 
    *   显示当月的打卡情况。
    *   **颜色深浅**代表当天的总学习时长 (0h 灰色 -> 6h+ 深绿)。
    *   点击月份两侧的箭头 `< >` 可以切换月份。
*   **最近7天趋势 (左侧上)**: 
    *   显示过去7天的总学习时长柱状图。
*   **科目分布 (左侧下)**: 
    *   显示本周（或根据日志累计）的各科目投入比例。
    *   颜色与上面的表格对应。

#### 4. 图片插入指南

**单张图片**:
```markdown
![我的笔记](/img/gwy/note.jpg)
```

**多图并排 (推荐)**:
使用我们封装好的组件，可以让 2-3 张图片完美并排显示。

```liquid
{% raw %}
<!-- 两张图 -->
{% include gwy_image_row.html 
    img1="/img/gwy/pic1.jpg" 
    img2="/img/gwy/pic2.jpg" 
%}

<!-- 三张图 -->
{% include gwy_image_row.html 
    img1="/img/gwy/pic1.jpg" 
    img2="/img/gwy/pic2.jpg" 
    img3="/img/gwy/pic3.jpg" 
%}
{% endraw %}
```
